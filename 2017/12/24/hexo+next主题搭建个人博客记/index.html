hexo+next主题搭建个人博客记hexo+next主题搭建个人博客记

发表于 2017-12-24

hexo是一个用来生成静态界面的框架，使用hexo搭建静态网页博客，实际上是使用hexo将本地博客内容渲染为静态html页面。运行hexo server可以通过本地4000端口访问这个静态网页，通过将静态网页提交到github pages个人主页后就可以通过https://gethub_name.github.io来访问你的静态网站了

基础环境配置

在开始之前，先安装一些过程中必要的组件。

  1. 安装 git bash 或者 cygwin，需要用到 git 提交且用到一些命令，所以 linux 风格的shell还是很必要的；
  2. 安装 node，windows直接下载 node 安装程序，安装好后打开shell使用命令node -v，查看一下node版本验证安装是否成功；
  3. 安装 hexo ，打开shell使用命令npm install -g hexo安装hexo

本文使用的环境如下：

  • Windows 10
  • cygwin
  • git version 2.15.1
  • node version v8.9.3
  • hexo version 3.4.4

初始化博客并安装 next 主题

安装hexo并初始化博客

在hexo安装的地方新建目录hexo，进入目录后使用命令npm install -g hexo安装hexo，并初始化博客

安装next主题

在hexo目录位置下载next主题
git clone https://github.com/iissnan/hexo-theme-next themes/next

与所有 Hexo 主题启用的模式一样。 当 克隆/下载 完成后，打开 hexo配置文件， 找到 theme 字段，并将其值更改为 next 。

theme: next
